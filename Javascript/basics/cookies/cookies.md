1.) What are cookies ?
Answer - 
a.) Cookies are used to store the information on the webpages.
b.) Cookies store value in the form of key-value pair and it holds around 4kb of data
c.) When the user login into the application server uses the set cookie http header in the response to set a cookie with the unique session identifier.
When user make an api request cookie will sent an http header request by which server will identify who the user is.
